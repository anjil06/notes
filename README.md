# 📝 Flutter Notes App

A simple and clean Notes application built using **Flutter** with **SQLite (sqflite)** for local data storage.

---

## 🚀 Features

* ➕ Add new notes using Floating Action Button
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 💾 Store data locally using SQLite
* 📄 Display notes in a clean list view
* 📭 Show "No Notes Yet" when empty
* 📱 Responsive UI with Bottom Sheet input

---

## 🛠️ Tech Stack

* **Flutter** (UI Framework)
* **Dart** (Programming Language)
* **sqflite** (Local Database)
* **path** (Database path handling)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/notes-app.git
```

2. Navigate to project folder:

```bash
cd notes-app
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

---

## 🧠 How It Works

* Notes are stored locally using SQLite database
* CRUD operations:

  * Create → Add note
  * Read → Fetch notes
  * Update → Edit note
  * Delete → Remove note
* Bottom Sheet is used for adding and editing notes
